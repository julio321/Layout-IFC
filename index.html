TECNOLOGIA EM SISTEMAS PARA INTERNET

Ser referência em educação, ciência e tecnologia na formação de proficionais comprometidos com o desenvolvimento de uma sociedade democrática,inclusiva, social e ambientalmente eqilibrada!

Metodologia

Aulas práticas e dialogadas desenvolvidas principalmente em laboratório uso de projetor multimídia e quadro branco.

Os Institutos Federais de Educação Ciência e Tecnologia constituem um novo modelo de instituição de educação profissional e tecnológico que visa responder de forma eficaz ás demandas crescentes por formação profissional e conhecimento científicos de suporte aos arranjos produtivos locais!

Objetivos do Curso!

Formar profissionais para o desenvolvimento de aplicações para internet no interesse da sociedade Desenvolver o raciocínio lógico-matemático obter formação em desenvolvimento de sofware para ambiente internet,intranet e móvel trabalhar em programação de computadores e em engenharia de software projetar e implementar sistemas de computação. O curso Superior de Tecnologia em Sistemas para Internet é um curso tecnológico de graduação com ênfase em desenvolvimento de sistemas de software para internet disciplinas específicas da área como tecnologias Web linguagens de programação engenharia de software redes de computadores, banco de dados, arquitetura de computadores e sistemas operacionais familiarizam os alunos com as questões ligadas ao desenvolvimento de sistemas de software para internet!
